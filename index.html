tmdb-java

A still incomplete Java wrapper around the TMDb v3 API, based upon JakeWharton/trakt-java/.

Pull requests (e.g. support for more API endpoints, bugfixes) against dev are welcome!

Usage

// Create an instance of ServiceManager once
ServiceManager manager = new ServiceManager();
manager.setReadTimeout(10000);
manager.setConnectionTimeout(15000);
manager.setApiKey("yourapikey");

// Build your query
Movie movie = manager.moviesService().summary(123).fire();

Using with Android

Because some device manufacturers shipped a non-hidden version of GSON you should replace the required GSON jar with a version with the package name customized, to e.g. myjson. For details see this [GSON help page][3] and the myjson branch (especially the libs folder) for an example on how to do this.
