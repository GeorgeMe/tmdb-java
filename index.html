tmdb-java

A Java wrapper around The Movie Database API v3. Pull requests for more endpoints welcome!

View project onGitHub

Dependencies

Releases are packaged without dependencies. Please add them as you see fit, e.g. for Maven:

<dependency>
  <groupId>com.squareup.retrofit</groupId>
  <artifactId>retrofit</artifactId>
  <version>(insert latest version)</version>
</dependency>

or for Gradle:

compile 'com.squareup.retrofit:retrofit:1.2.2'

Usage

// Create an instance of the service you wish to use
// you can keep this around
Tmdb tmdb = new Tmdb();
tmdb.setApiKey("yourapikey");
MovieService movieService = tmdb.movieService();

// Call any of the available endpoints
Movie movie = movieService.summary(550);
Trailers trailers = movieService.trailers(550);
